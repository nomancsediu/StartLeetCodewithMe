# The Transparent Compiler

An interactive web application that visualizes how a compiler works internally, built with Django and modern web technologies.

## Features

- **Lexical Analysis**: See how code is broken down into tokens with animated, color-coded visualization
- **Syntax Analysis**: View the Abstract Syntax Tree (AST) generation in real-time
- **Output/Evaluation**: Watch the code execution with typewriter-style result display
- **Real-time Updates**: Code changes are instantly reflected in visualizations
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## Technology Stack

- **Backend**: Python Django with Django REST Framework
- **Frontend**: HTML5, Tailwind CSS, Vanilla JavaScript
- **Parsing**: Custom lexer and parser implementation
- **Styling**: Material Design icons, Space Grotesk font
